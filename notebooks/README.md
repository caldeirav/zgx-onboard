# Notebooks

This directory contains Jupyter notebooks for experiments and demonstrations.

## Structure

- `inference/` - Model inference examples
- `agents/` - AI agent demonstrations
- `tuning/` - Model tuning and optimization examples
- `reinforcement_learning/` - RL experiments

## Running Notebooks

1. Make sure you have Jupyter installed:
   ```bash
   pip install jupyter jupyterlab
   ```

2. Start Jupyter Lab:
   ```bash
   jupyter lab
   ```

3. Navigate to the notebook you want to run.
