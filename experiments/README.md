# Experiments

This directory contains experimental code and scripts for various AI experiments.

## Structure

- `inference/` - Inference experiments
- `agents/` - Agent experiments
- `tuning/` - Hyperparameter tuning experiments
- `reinforcement_learning/` - RL experiments

## Running Experiments

Experiments can be run directly as Python scripts or through experiment tracking tools like MLflow.

Example:
```bash
python experiments/inference/run_inference.py --config configs/default_config.yaml
```
