# Examples

This directory contains example scripts demonstrating how to use the ZGX Onboard package.

## Available Examples

Examples will be added here as the project develops. Each example will demonstrate:
- Model inference
- Agent training
- Model tuning
- Reinforcement learning

## Running Examples

```bash
# Activate virtual environment
source venv/bin/activate

# Run an example
python examples/inference_example.py
```
